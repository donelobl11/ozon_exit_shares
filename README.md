


1. Установить python (если еще не установлен на компьютер)

2. Открыть консоль powershell

3. Перейти в расположение папки (там где файл main.py лежит) --- cd C://Путь к папке

4. Войти в гитхаб, войти в репозиторий нажать зеленую кнопку Code, скопировать ссылку для клона репозитория.
Пример ссылки https://github.com/example/worker.git
Вставить в консоль и нажать энтер

5. Создать виртуальное окружение командой --- python -m venv venv

6. Активировать виртуальное окружение командой \venv\Scripts\Activate.ps1 в консоли powershell. Пример "d:\Путь к папке\venv\Scripts\Activate.ps1"

7. Установить зависимости из проекта командой --- pip install -r requirements.txt

8. Запустить проект командой main.py
